# Chord PDF

Renders text files of chord charts as PDFs.

Install with `npm install -g @anbcodes/chordpdf`

Usage: `chordpdf <input file> <output file> --key [key]`

Example input file

```
Key: C
Title: Example
Author: Example Author
BPM: 71

# Verse 1
1              4
 Some lines of a song
4                   5
Next line, and again
6m   2m
More chords
4   5       1
And finally

# Interlude
(1) / / / | / / / /
```
